# Aplicação de E-mails com GraphQL

## Resumo
Uma pequena aplicação de emails utilizando graphql com lógica para:
1. Usuários
2. Emails

Uma adição a dificuldade, é que um usuário que envia o email pode deletar o email caso o destinatário ainda não o tenha visto.

## Tecnologias usadas
+ GraphQL
+ Apollo Server
+ Express
+ Prisma ORM
+ SQLite
+ cors
+ bcrypt
+ jsonwebtoken
+ reflect-metadata
+ dotenv

## Como rodar o projeto
```bash
npm i
npm run dev
```
Ao rodar o projeto no terminal, rodar em máquina local e clicar neste [Link](http://localhost:4000) para acessar o **Apollo Server**.
